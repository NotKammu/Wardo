# Project Title
Wardo: Digital Wardrobe and Fashion Stylist

# Project Overview
Wardo is an intelligent fashion platform that helps users manage, explore, and enhance their style. Users can upload images of their clothing to create a personalized digital wardrobe. Based on occasions, weather, and physical features, Wardo recommends the best outfit combinations. Additional features include outfit planning.

# Technical Details
Python, Flask
SQlite, SQQLAlchemy
OpenCV, numpy
Flask CORS
